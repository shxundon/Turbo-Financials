🛠️ COMMON COMMANDS
══════════════════════════════════════════════════════════════════════════════

START APPLICATION:
  .\run.bat              (Windows)
  ./run.sh               (macOS/Linux)

VIEW SUBMISSIONS:
  python manage.py summary              # Show stats
  python manage.py recent               # Last 5
  python manage.py recent -n 10         # Last 10
  python manage.py view 0               # First submission

EXPORT DATA:
  python manage.py export --csv         # To Excel
  python manage.py export --json        # To JSON
  python manage.py export --csv --type tax_information

MANAGE DATA:
  python manage.py delete-old --days 30 # Delete old

TEST EVERYTHING:
  python test.py                        # Test all endpoints

VIEW LOGS:
  type logs\app.log                     # Windows
  cat logs/app.log                      # macOS/Linux

🔐 DATA SECURITY
══════════════════════════════════════════════════════════════════════════════

What IS Logged:
  ✓ Name, email, address, phone
  ✓ Filing status, dependents
  ✓ Income amounts
  ✓ Deduction amounts
  ✓ Bank name, account type
  ✓ Timestamps

What is NOT Logged (Masked):
  ✗ Full SSN (presence only)
  ✗ Full account numbers (last 4 only)
  ✗ Full routing numbers (last 4 only)

All data is stored locally in JSON files. No cloud, no external servers.
You have complete control over your data!
